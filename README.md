# OpenTRS-Core

Open Thermal Reference Space Core

An open-source radiometric framework for thermal imaging.

## Goals

- Accurate radiometric decoding
- Temperature reference workflow
- Scientific validation
- Creator-oriented thermal rendering
- Open architecture
